# Prompt Library — UI 升级设计规格文档

> **设计方向:** Editorial + Refined Minimalism（编辑/杂志风 + 精致极简）
> **上一轮:** iOS 26 设计语言重构 (2026-05-26-ios26-design-refactor)
> **本轮目标:** 在 iOS 26 基础上深化排版、色彩、动效，从「能用好看」升级为「有设计记忆点」

---

## 设计理念

### 设计哲学

```
克制但不沉闷，排版主导视觉层级，
单一强调色 + 大面积中性色，
卡片间通过透明度 + 比例差异制造节奏，
微妙线条和边框替代粗重阴影。
```

### 从 iOS 26 → Editorial + Refined

| 维度 | iOS 26 设计 | 本轮升级 |
|------|-----------|---------|
| 色彩系统 | 紫/黄双色 + 5 分类色 | 蓝靛 primary + 暖石 neutral + 克制分类色板 |
| 卡片材质 | 统一 0.72 透毛玻璃 | 三级透明度体系 (0.85 / 0.72 / 0.55) |
| 排版 | SF 字重体系 | 强化字重对比 + 留白节奏 + 不对称版面 |
| 动效 | 无 | 交错入场 + 弹性微交互 |
| 背景 | 纯色 #F8F8FA | subtle 噪点纹理 (light) / 纯黑 (dark) |

### 间距体系 (4px 基础网格)

| Token | 值 | 用途 |
|-------|---|------|
| `xs` | 4 | 元素内微间距 |
| `sm` | 8 | 图标-文字间距 |
| `md` | 14 | 列表项间距 |
| `lg` | 20 | 卡片内边距 |
| `xl` | 28 | 区块间距 |
| `xxl` | 40 | 大区块分隔 |

**规则:** 相邻元素用 sm/md，相邻区块用 lg/xl，页面级用 xxl。

## 毛玻璃实现策略 (React Native)

由于 RN 不支持 CSS `backdrop-filter: blur()`，采用视觉等效方案：

1. **多级透明度** — `card`(0.85) / `cardSecondary`(0.55) / `cardElevated`(0.96) 建立层次
2. **极细边框** — `0.5px separator` 弱边缘感
3. **柔和阴影** — `Shadow.md/lg` 替代平面排列
4. **背景纹理** — light 模式 subtle noise 增加质感
5. **contentInset 间距** — 卡片与背景之间保留足够面积透出底色
